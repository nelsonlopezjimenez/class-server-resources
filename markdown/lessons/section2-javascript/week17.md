## Assignment 17: Recipe Manager

For this assignment you will create a small application that stores recipes. Each recipe will be represented by an object, and you’ll use an array to hold multiple recipes and loops to iterate over the data.

**Instructions:**

1. Create a new .js file in your section 2 Assignment folder.
1. Open that file in VSCode.
1. Create pseudocode for your application before you start coding.
1. Add Code to meet the criteria below.
1. Don't forget to properly comment your code/file.
1. Submit the file to assignment 17.

```javascript
// ** Create a Recipe object here **

// ** Create an array to hold the recipe objects here **

// ** Create a function to add a new recipe to the array **

// ** Create a function to display the recipes in the console **

// ** Create a function to remove a recipe from an array. **

// This is the main program loop.  When the code is executed this function will continuously show a prompt request to the user asking them what they want to do:
// 1) Display all the recipes
// 2) Add a new recipe
// 3) Remove an recipe
// 4) Exit the program (use a break statement to break out of the loop)
// 5) If any thing is typed that is not 1-4 remind user of available options
```

**Required Functionality:**

- An array that holds 3-5 recipe objects
  - Each recipe should have a property to store its:
    - `name`: (String) - The name of the recipe.
    - `ingredients`: (Array of Strings) - An array of ingredients.
    - `instructions`: (String) - The recipe instructions.
    - `img`: (String) - stores the relative path to a fake image file (use ./image/fileName.png)
- A add function that:
  - takes an array as a parameter
  - Creates a new `Recipe` having the same properties as above
  - Adds the new `Recipe` object to the end of the `recipes` array.
  - Returns the updated `recipes` array.
- A display all function that:
  - Takes the `recipes` array as a parameter.
  - Uses a `for...of` loop to iterate through each `Recipe` object in the array.
  - For each `Recipe` object, prints the recipe’s name, ingredients, and instructions to the console.
- A display one recipe by name function that:
  - Takes the `recipes` array as a parameter.
  - Uses a `for` loop to iterate through each `Recipe` object in the array.
  - a conditional statement that checks if the name searched for is in the array. If it is it prints out only that recipe’s name, ingredients, and instructions to the console. If that item is not found it prints "Recipe with name `NAME` not found."
- A remove one function that:
  - Randomly selects a number between 0 and the number of elements in the array.
  - Removes selected recipe from from the array
  - Returns the new array
- An update function

  - Randomly selects a number between 0 and the number of elements in the array.
  - Updates that recipes name to be `Chunky Monkey Icecream`
  - Returns the updated array

- Console logs for:
  - "Before Changes Made: [Array]" - use display all function
  - "Showing recipe `recipe-name`: [Recipe]" - use display one function
  - "After an item is added: [Array]" - use add one function
  - "After a random item is removed: [Array]" - Use remove one
  - "After a recipe has been updated: [Array]" - use update function

**Extra Credit:**

- **HTML Integration:** Create an HTML page with the following elements:
  - An input text box for the recipe name.
  - Input fields for ingredients (use comma delineation and and split the string to create an array).
  - Input field for the instructions.
  - Four buttons:
    - "Add Recipe"
    - "Update Recipe"
      - This will update a random recipe with the name in the text box
    - "Display All Recipes"
    - "Display One Recipes"
      - Will display one recipe where the recipe name matches text box, or "No recipes found"
  - A UL element to display the recipe items.
  - When the "Add Recipe" button is clicked, add the recipe data to the JavaScript array and update the UL element.
  - When the "Display Recipes" button is clicked, display the recipes in the UL element.
