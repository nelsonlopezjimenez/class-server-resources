# Week 17: Objects and Loops

## Learning Objectives

By the end of this week, students will be able to:

- Identify and work with Objects
- Properly use object methods
- Use loops to repeat code
- Know when to us a Do, While, For, For Of, For In loop
- Understand structure of a loop, increment/decrement and conditions.
