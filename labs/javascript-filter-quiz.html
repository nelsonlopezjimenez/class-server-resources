Given an array of words, filter and return only words that have more than 5 characters.

const words = ["apple", "banana", "cat", "dog", "elephant", "fish", "giraffe"];

Your task: Return only words with length > 5: ["banana", "elephant", "giraffe"]

💡 Hint:

Access the length of a string using .length property

Example: words.filter(word => word.length > 5)

Filter Students by Grade

Given an array of student objects, filter and return only students who have a grade of 85 or higher.

const students = [ { name: "Alice", grade: 92 }, { name: "Bob", grade: 78 }, { name: "Charlie", grade: 85 }, { name: "Diana", grade: 95 }, { name: "Eve", grade: 72 } ];

Access object properties using dot notation: student.grade

Filter Products by Price and Availability

Filter products that are both in stock AND cost less than $50.

const products = [ { name: "Laptop", price: 999, inStock: true }, { name: "Mouse", price: 25, inStock: true }, { name: "Keyboard", price: 45, inStock: false }, { name: "Monitor", price: 200, inStock: true }, { name: "USB Cable", price: 10, inStock: true }, { name: "Headphones", price: 60, inStock: true } ];

Filter Users by Age and Location
