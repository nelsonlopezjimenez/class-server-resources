# Weekly Schedule and expectations

I added an "Introduction to [Subject]" to each section. This is where section specific details can go.

Each week is structured the same regardless of section:

Day 1 - 3 introduces new concepts each day
Day 1 - 3 has 1-3 labs that are short code examples that need to be turned in daily
Day 4 is for week review, a quiz or two (10-20 questions each), and an assignment due first day of next week
