# Weekly Schedule and expectations

I am making each section 10 weeks long. The reason is that some quarters are 10, 11, or 12 weeks. On average there are 44 instructional weeks a year, but 8 of those are either 2 or 3 day weeks. Plus add in the days missed due to DOC safety musters, codes, etc and 40 weeks of 4 days or 160 days is closer to the actual days in class rather than the 48 x 4 of 192 days.

I added a week 0 to the the first section that was essentially the old week 1, this only has 3 days (as the first week always starts on a tuesday)

Week 1 starts day 1 with Instructional HTML learning through freeCodeCamp

Each week is structured the same regardless of section:

Day 1 - 3 introduces new concepts each day
Day 1 - 3 has 1-3 labs that are short code examples that need to be turned in daily
Day 4 is for week review, a quiz or two (10-20 questions each), and an assignment due first day of next week

Weekly breakdown

Week 1-5 HTML

Week 1: Understanding HTML Attributes, HTML Boilerplate, HTML Fundamentals
Week 2: HTML SEO, Images, Links
Week 3: Semantic HTML, Nuanced Text and Time, Specialized Semantic Elements
Week 4: Working with Forms, Working with Tables, Working with HTML
Week 5: HTML Midterm Assignment (no CSS) and midterm Exam

Week 6-10 CSS

Week 6: Basic CSS, Design
Week 7: Absolute and Relative Units, Pseudo Classes, Colors
Week 8: Styling Forms, Box Model, Flexbox
Week 9: Typography, Responsive Design, Grid
Week 10: Peach Cobbler Responsive Website (day 1 mobile, day 2 tablet, day 3 desktop) and Final Exam

Week 11-20 Javascript

Week 11: Variables and Strings
Week 12: Booleans and Numbers
Week 13: Functions
Week 14: Arrays, Objects
Week 15: Loops, Midterm Assignment and Exam
Week 16: Higher order functions, callbacks, DOM Manipulation
Week 17: Debugging, Form Validation
Week 18: Maps and CRUD
Week 19: Recursion and Async
Week 20: Final Assignment and Exam

Week 21-30 Node.js (Backend)

These weeks will need to be manually created as there is nothing available on fCC as of yet. Odin Project has some, but contains many links to outside resources.

Week 31-40: React (Frontend)

Week 31: Intro to JS Libraries and Frameworks, Working with Data in React
Week 32: React State, Hooks and Routing
Week 33: Performance
Week 34: Testing
Week 35: Build Tic Tac Tow Game and Midterm Exam
Week 36-40: Final Project (Recipe Site from Scratch not from a pre made site that is 90% done...) and Final Exam
